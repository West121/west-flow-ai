# PLM v2 Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 把当前 PLM 从基础审批闭环升级为具备业务台账、生命周期、详情工作区和 AI 摘要增强的 `PLM v2`。

**Architecture:** 保持现有三类业务域 `ECR / ECO / 物料主数据变更` 不变，在现有表模型上扩字段、补生命周期接口、补工作台汇总和统一详情区块。前端以工作台、列表、详情三层重构，AI 工具保持兼容键名但增强返回内容。

**Tech Stack:** Spring Boot, MyBatis, Flowable, React, TanStack Query, react-hook-form, zod

# PLM 企业级路线执行计划

## 目标

在当前 `PLM v4` 基础上，继续推进到完整企业级 PLM 平台骨架。

## 阶段拆分

### Phase 1：v5 对象与权限基础

交付：

- BOM 主表 / 行表
- 文档/图纸扩展对象模型
- 对象级 ACL
- 域级 ACL
- PLM 专属角色矩阵
- BOM / 文档 / 图纸工作区

验证：

- 单据可挂接真实 BOM / 图纸 / 文档对象
- 不同角色看到的对象范围正确
- 前端详情可展示 BOM 层级与对象关系

### Phase 2：v6 版本与实施深化

交付：

- 基线表 / 配置快照表
- BOM 差异结构比较
- 实施任务模板
- 依赖关系
- 验证证据链

验证：

- 能按基线查看版本集
- 能比较变更前后 BOM
- 能解释为什么某张单据不可关闭

### Phase 3：v7 管理驾驶舱与外部边界

交付：

- SLA/超期/返工分析
- 组织与产品线驾驶舱
- 报表导出
- 文档预览与受控版语义
- 外部 PDM/CAD/ERP 集成边界定义

验证：

- 管理看板能反映瓶颈与风险
- AI 能回答经营分析与执行瓶颈问题

## 当前立即执行

下一轮直接进入 `Phase 1`：

1. 设计并落库 BOM 与文档对象结构
2. 设计 ACL 与角色矩阵
3. 前端补对象工作区和 BOM 展开
4. 后端补对象级授权拦截
